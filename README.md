## 🔥 What Is GrokForge?

GrokForge is a platform where entrepreneurs build businesses that **run themselves**. Grok AI handles:

- **Market Intelligence** — Real-time competitive analysis, trend detection, opportunity scoring
- **X/Twitter Engine** — AI-generated content, growth strategies, audience automation
- **DOGE Efficiency Scoring** — Find waste, kill it. Elon's "delete the process" methodology
- **Autonomous Agents** — Grok-powered workers for sales, support, content, trading
- **Business Analytics** — LTV/CAC ratios, churn prediction, revenue forecasting

## ⚡ Quick Start

```bash
git clone https://github.com/eugenemcmillian220-ui/grokforge-ai.git
cd grokforge-ai
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Connect Grok

1. Get an API key at [console.x.ai](https://console.x.ai)
2. Go to Settings in GrokForge
3. Paste your xAI API key
4. Unlock real-time Grok intelligence

> **Works in demo mode without an API key** — mock responses showcase all features.

## 📊 Features

### 🧠 Command Center
Direct chat interface with Grok. Ask it to analyze markets, generate business plans, optimize operations, or draft X content.

### 🚀 Business Dashboard
Track all your autonomous businesses. Revenue, agent count, DOGE efficiency scores, X follower growth.

### 🤖 Grok Agents
6 autonomous AI workers: DesignGrok, MarketGrok, SupportGrok, DataGrok, SalesGrok, TradeGrok.

### 📈 Market Intelligence
Real-time signals: opportunities, threats, trends, disruptions. Confidence scoring and impact assessment.

### 💬 X Engine
Generate X/Twitter posts in three styles: witty, informative, provocative. Predicted engagement scoring.

### ⚡ DOGE Score
Department of Government Efficiency methodology applied to your business. Waste audit and savings projections.

### 📊 Analytics
Revenue trajectory, MRR growth, LTV/CAC ratios, churn rates, NPS scores.

### 🔒 Security
API security scoring, encryption status, auth integrity, agent sandboxing.
